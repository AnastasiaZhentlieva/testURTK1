Техническое задание "Простой калькулятор"

1. Назначение
  
   Простое приложение для базовых математических вычислений.

2. Функции

2.1. Консольная версия

· Сложение, вычитание, умножение, деление
· Меню выбора операций
· Проверка деления на ноль
· Постоянная работа до команды выхода

2.2. Версия с GUI

· Цифровые кнопки 0-9
· Кнопки операций: +, -, *, /
· Кнопки: = (равно), C (очистка), . (точка)
· Поле для отображения чисел и результатов

3. Технические требования

· Язык: C#
· GUI: Windows Forms (для графической версии)
· .NET Framework 4.5+

4. Обработка ошибок

· Проверка ввода чисел
· Защита от деления на ноль
· Сообщения об ошибках без падения программы

5. Критерии готовности

· Все операции работают правильно
· Нет ошибок при вводе неверных данных
· Удобный интерфейс
· Стабильная работа
